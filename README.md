# stocks-app

## Description

ReactJS project using Typescript and Redux

</br>

## Installation

```
npm install
```

you need to get an api key from api.twelvedata

## ENV vars

```
REACT_APP_API_HOST=http://localhost:3000
REACT_APP_EXTERNAL_BASE_URL=https://api.twelvedata.com
```
